CONTEXT:
Backend recently switched from DALL-E to Unsplash API for article images. The backend now provides these fields for each article:
- image_url (the Unsplash photo URL)
- image_credit (text: "Photo by {Name} on Unsplash")
- image_credit_url (photographer's Unsplash profile URL)
- image_photographer (photographer name only)

Every photo displayed on the website MUST include attribution with:
1. Photographer name (linked to their Unsplash profile)
2. Text "on Unsplash" (linked to https://unsplash.com)
3. Displayed near/below the image
